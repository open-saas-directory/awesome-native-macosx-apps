### What Makes an App "Native"?

Apps on this list are:

- ‚úÖ Built with native macOS frameworks (Swift, SwiftUI, AppKit, Objective-C)
- ‚úÖ Lightweight and resource-efficient
- ‚úÖ Follow Apple's Human Interface Guidelines
- ‚úÖ Feel fast and responsive
- ‚ùå NOT Electron or web-wrapper apps
- ‚ùå NOT resource-heavy cross-platform apps

## Contributing

We welcome contributions! Please read our [Contributing Guidelines](CONTRIBUTING.md) before submitting.

### How to Contribute

1. **Fork** this repository
2. **Add** your app to the appropriate category in alphabetical order
3. Follow the format: `[App Name](link) - Brief description. \`License/Price\``
4. **Verify** the app is truly native (Swift/SwiftUI/AppKit)
5. Create a **Pull Request**

### App Requirements

Apps must meet ALL these criteria:

- ‚úÖ Built with native macOS technologies
- ‚úÖ Lightweight and performant
- ‚úÖ Actively maintained (updated within the last 2 years)
- ‚úÖ Available for download
- ‚úÖ No malware/spyware

### What NOT to Include

- ‚ùå Electron apps
- ‚ùå Web wrappers
- ‚ùå Abandoned projects (no updates in 2+ years)
- ‚ùå Apps with malware/suspicious behavior
- ‚ùå Cross-platform apps with poor macOS integration
